What is the task
why is it important/interesting?

Our task is to predict the result of an incoming soccer match(Win, Draw or Lost) in Spanish League(La Liga) for Home team based on attributes of both two teams, along with other factors such as weather, and live sports betting odds from Beat365. Predicting the outcomes of sport events represents a natural application for machine learning. The precise game outcome can also be critical to soccer fans and bettors as a source of advice. As a consequence, using machine learning technology to handle big data of soccer statistics is also a good way to test ourselves how well we have learnt in this course.

Features we used

We are working on an open source European Soccer Database with 25k+ matches, players & teams attributes for European Professional Football, and FIFA data API with players and teams attributes together, reprocess with SQL script to generate a well structed data frame for our ML model. We decide to use last 10 years match statistics of La Liga with:

Weather, Home Team, Away Team, HomeRank, AwayRank, WinRateHome, WinRateAway, LastFiveGameWinRateHome, LastFiveGameWinRateAway, Beat365H(Home Win), Beat365D(Draw), Beat365A(Away Win)

as attributes for each game.

A Neural Networking Model
Implemented by R

We have implemented a BP neural networking model by R language. It has three layers:
one input layer
one hidden layer with activiation function ReLu
one output layer with activiation function Maxout.
