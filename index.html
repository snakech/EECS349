What is the task
why is it important/interesting?

Our task is to predict the result of an incoming soccer match(Win, Draw or Lost) in Spanish League(La Liga) for Home team based on attributes of both two teams, along with other factors such as weather, and live sports betting odds from Beat365. Predicting the outcomes of sport events represents a natural application for machine learning. The precise game outcome can also be critical to soccer fans and bettors as a source of advice. As a consequence, using machine learning technology to handle big data of soccer statistics is also a good way to test ourselves how well we have learnt in this course.

Features we used

We are working on an open source European Soccer Database with 25k+ matches, players & teams attributes for European Professional Football, and FIFA data API with players and teams attributes together, reprocess with SQL script to generate a well structed data frame for our ML model. We decide to use last 10 years match statistics of La Liga with:

Home Team Lineup, Away Team Lineup, HomeRank, AwayRank, WinRateHome, WinRateAway, LastFiveGameWinRateHome, LastFiveGameWinRateAway, The difference between Beat365H(Home Win) and Beat365A(Away Win)

as attributes for each game.

A Neural Networking Model
Implemented by R

We have implemented a BP neural networking model by R language. It has three layers:
one input layer
one hidden layer with activiation function ReLu
one output layer with activiation function Maxout.

The top figure shows the learning curve of training set, the bottom one shows the learning curve of testing set

The NN model can archive around 80% accuracy of training, but testing performance dropped down to 55%

Findings and Conclusion

ZeroR provides us a baseline of the accuracy we could achieve through learning. Obviously, our task is far away from "If-else-then" problem and it's much more complicated than simply splitting/filtering examples according to some attributes and then getting the right result. "K-Nearest-Neighbor" model in our task could be considered as "If home team is close enough to K other teams, then we can predict result from other similar team's exsit results". But it has a pain point which is really difficult to figure out and choose the proper "K", we have tried K = 1, 3 and 7 which 7NN has the best performance.

Therefore, we choose KNN as the basic model for our predictor.

For our Neural Networking model we implemented, it has four layers, the first layer is input layer, with 24 feature values from dataset. the second hidden layer has 256 cells, with 0.2 dropout to avoid overfitting, the activiation function is ReLu, which have better efficiency and performance than Sigmoid. The third hidden layer has 64 cells, with 0.1 dropout, activiation function is ReLu as well. The last output layer we choose Maxout to be the output function, we think Maxout is suitble to deal with multi-classification problem, which we need a prediction of Win, Draw and Loss.
